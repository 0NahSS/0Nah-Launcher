Mémoire Ram

Information

Cette page permet de configurer la quantité de mémoire RAM que votre système d'exploitation va allouer à Paladium.

Plus cette quantité est importante, plus votre jeu sera stable et aura de bonnes performances.
Toutefois, même si vous en avez beaucoup de disponible sur votre ordinateur, il est inutile d'allouer une très grande quantité de RAM à Paladium.

- RAM Maximum: Indique la quantité totale de RAM à allouer à Paladium. Il est recommander de ne pas dépasser 50% de la quantité totale de RAM présente sur votre PC. Si votre ordinateur a plus de 8Go de RAM, nous recommandons d'allouer 3Go. Si votre ordinateur a 4Go ou moins, n'allouez pas plus de 2Go.

- RAM Minimum: Indique la quantité minimale de RAM allouée au démarrage de Paladium. Il n'est pas recommandé de mettre plus de 3Go pour cette valeur. La valeur typique est de 1Go.

Java exécutable

Information

Cette page permet d'indiquer à hariona l'emplacement de l'exécutable Java.

Le launcher utilise une version embarquée de Java (Java 8 update 51), si celui-ci n'est pas déjà installé sur votre ordinateur. <% if(process.platform === 'win32') { %> Cette version est disponible dans le répertoire %appdata%/.hariona/runtime.

Toutefois, vous pouvez spécifier votre propre exécutable.
Celui-ci se trouve très souvent à cet emplacement:
C:\Program Files\Java\jre1.8.0_XXX\bin (avec XXX le numéro du build de la version)

Deux versions de l’exécutable sont disponible dans ce répertoire:
- java.exe
- javaw.exe

Par défaut, utilisez javaw.exe. <% } %>
